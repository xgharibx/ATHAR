## ما تم التحقق منه ووُجد صحيحاً (لا يحتاج إصلاح)

| الموضوع | السبب |
|---|---|
| توسيع بطاقة أسماء الله | تصميم inline expand (ليس dialog) |
| أخطاء Supabase في الاختبار | لا إنترنت في بيئة الاختبار — سلوك متوقع |
| مسار `/adhkar/ruqyah` يعطي 404 | المسار الصحيح هو `/ruqyah` |
| `DIV.flex` مع overflow في الصفحة الرئيسية | داخل `overflow-x-auto no-scrollbar` — مقصود |
| Ctrl+K لا يعمل عبر Playwright | محدودية synthetic event — يعمل بالنقر عبر مؤشر الشاشة |
