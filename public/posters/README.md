# Poster background images

Optional background photos used by the “Share as photo” poster generator.

Put images here:

- `public/posters/roses/01.jpg`
- `public/posters/roses/02.jpg`
- `public/posters/system/01.jpg`

If files are missing, the app falls back to a generated gradient background.

Notes:
- Use **JPG** (recommended) or PNG.
- Best size: at least ~2000px on the long edge.
- Keep images high-contrast friendly (the app will apply a dark overlay).
